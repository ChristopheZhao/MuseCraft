# MuseCraft Brand Snippets

This folder provides lightweight, production‑ready brand assets for quick external use.

## Files

- `musecraft-mark.svg`: Monogram icon (M + C) with gradient stroke. Use as app icon, avatar, or favicon source.
- `musecraft-logo.svg`: Horizontal logo (icon + wordmark) for light backgrounds.
- `musecraft-logo-dark.svg`: Reversed logo for dark backgrounds.

## Color Palette

- Primary Gradient: `#6C5CE7` → `#00D4FF`
- Accent: `#10B981`
- Ink (dark text): `#0B1020` ~ `#111827`
- Surface: `#F5F7FB`

## Usage

- Minimum clear space: height of the icon spark around the logo.
- Minimum size: mark ≥ 24px, full logo ≥ 160px width.
- Do not stretch or skew. Keep aspect ratio.
- On dark backgrounds, prefer `musecraft-logo-dark.svg`.

## Tagline (optional)

```
AI 多智能体动漫创作平台
```

You can uncomment the subline in `musecraft-logo.svg` if needed.
