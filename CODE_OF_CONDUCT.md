# Code of conduct

We expect project interactions to be respectful, technically constructive and focused on the work.

Acceptable participation includes giving actionable feedback, acknowledging uncertainty, respecting different experience levels and protecting private data. Harassment, discrimination, threats, sexualized attention, doxxing and deliberate disruption are not acceptable.

Maintainers may edit or remove content and restrict participation when behavior harms contributors or the project. Report conduct concerns through the same private channel described in [SECURITY.md](SECURITY.md); do not publish personal information in an issue.
